About

WireIt is an open-source javascript library, that allows you to create cool wires like Yahoo Pipes. You can see it in action in the Planar Game, in jsBox, or in the examples below. It currently supports Firefox 1.5+, Safari 2.0+, IE 7.0 and Opera 9+. Please report your issues with specific browsers in the forum. It uses the YUI library for DOM and events manipulation, and excanvas for IE support.
